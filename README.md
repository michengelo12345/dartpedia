Descrição do Projeto
O Dartpedia é um projeto desenvolvido utilizando a linguagem Dart com o objetivo de permitir a busca, consulta e exibição de informações de forma simples e organizada. O projeto tem como finalidade praticar conceitos de programação, consumo de dados e desenvolvimento de aplicações usando o Dart.

Como Executar o Projeto
Abra o terminal na pasta do projeto.
Execute o seguinte comando:
dart run bin/dartpedia.dart


Comandos Disponíveis
Exibir ajuda
dart run bin/dartpedia.dart help


Exibir versão
dart run bin/dartpedia.dart version
	
Buscar um termo
dart run bin/dartpedia.dart search "Flutter"


Exibir informações sobre um tópico
dart run bin/dartpedia.dart info "Dart"


Tecnologias Utilizadas
Dart
Terminal (CLI)
Objetivo Educacional
Este projeto foi desenvolvido com fins acadêmicos para praticar programação em Dart, utilização de argumentos via terminal e desenvolvimento de aplicações de linha de comando.
